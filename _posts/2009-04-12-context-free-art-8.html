やっと描けるようになった…

startshape main
background {h 65 sat .1}
tile {s 3.2}

rule main {
  4 * {h 90 x .8 y .8} cubes {h 320 sat .3}
}

rule cubes {
  2 * {y 1.6} CUBE {r 45}
}

rule CUBE {
  4 * {x .333 y .333 r 90} TOWER {s 1 .576 b .4}
}

rule TOWER {
  TRIANGLE {}
  TOWER {y .002 s .99 b .02}
}

オリジナルは、昔美術の授業で作った課題作品。二点透視図法を組み合わせたデザイン。
